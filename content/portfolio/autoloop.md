A plugin for agentic tools like Claude Code that enables AI agents to work independently until a task is completed, eliminating constant back-and-forth interaction.

**What makes it powerful:**

- Autonomous execution with self-correcting iterations
- Safety controls with configurable iteration limits
- Full transparency through git commits at each milestone
- Completion promises ensure genuine task completion
